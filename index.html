一、内部控制标准
1. 关企合作 (1)-(4) 赋分
(1) 海关沟通与人员信息告知
指定海关业务联系人，通过"中国海关信用管理服务平台"等方式与海关开展沟通与合作。企业相关人员在其他企业担任法定代表人或财务负责人的，须及时告知海关。
📖 须在信用管理服务平台注册；人员交叉任职须主动告知海关，防止信用风险传导。
📋 审核证据清单 📄 制度：
  • 关企沟通联系合作机制文件
  • 海关业务联系人任命文件
📝 记录：
  • "中国海关信用管理服务平台"注册截图及使用记录
  • 与海关沟通的会议记录/邮件/电话记录
  • 企业相关人员在外任职登记表
  • 告知海关交叉任职的书面通知
👤 面谈：
  • 联系人是否清楚沟通渠道和报告义务
💡 建立人员兼职登记表，每季度核查管理层在外任职情况，变化后5个工作日内告知海关。
(2) 高管掌握总体情况
法定代表人或关务负责人应当掌握企业涉及海关业务的总体情况，包括进出口业务、遵守海关监管规定、信用状况等，有效管控不规范情形。
📖 海关认证会约谈高管，验证其了解进出口总值、主要商品、信用等级、处罚记录等。
📋 审核证据清单 📄 制度：
  • 总经理/关务负责人岗位职责说明书
  • 公司组织架构图
📝 记录：
  • 关务运营简报/月报/季报（高管签收记录）
  • 高管参与关务会议记录
  • 不规范行为响应/整改签署记录
👤 面谈：
  • 约谈法人/关务负责人：进出口总值、主要商品、信用等级、近期处罚
(3) 关务人员岗位能力
关务人员、报关人员及进出口相关岗位人员应当熟悉本岗位相关的进出口业务及海关管理要求，及时处置不规范情形并有效整改。
📖 2026版新增。不要求精通全部法规，但须熟悉本岗位直接相关的要求。
📋 审核证据清单 📄 制度：
  • 各关务岗位职责说明书
  • 岗位能力矩阵/技能清单
📝 记录：
  • 培训计划、签到表、考试记录
  • 不规范情形处置及整改记录
👤 面谈：
  • 随机抽取关务人员，询问岗位相关海关法规
(4) 配合海关执法
积极配合或协助海关办理案件侦查、调查，以及开展稽查、贸易调查、核查等工作，并提供相关数据或资料。
📖 2026版新增。不仅不抗拒（底线），更要求主动协助。
📋 审核证据清单 📝 记录：
  • 历次海关稽查/核查配合记录及往来函件
  • 向海关提交数据/资料的签收记录
  • 海关稽查结论文书及后续整改记录
2. 单证复核 (5) 赋分
(5) 申报内容内部复核
明确制单、复核岗位工作流程和审核指引，在申报前有专门岗位人员对申报内容的真实性、完整性、准确性和规范性进行内部复核。
📖 "审核指引"是2026版新增。制单人≠复核人（四眼原则）。
📋 审核证据清单 📄 制度：
  • 《物料和设备进口/出口操作指引》
  • 报关单复核审核指引/检查清单
  • 制单和复核岗位SOP
📝 记录：
  • 抽查5-10份报关单复核记录（复核人签名、日期）
  • 确认制单人≠复核人
  • 删改单月度统计表
  • 差错报告及改进措施
🖥️ 系统：
  • 关务系统（易网通）复核流程截图
3. 单证保管 (6) 赋分
(6) 单证归档与保管
明确进出口单证、相关记录的归档流程和保管要求，按规定保存账簿、单证、记录等资料及海关核发的证书、法律文书等。
📋 审核证据清单 📄 制度：
  • 《进出口部文件归档工作指引》
  • 《印章管理程序》《证照管理办法》
📝 记录：
  • 抽查5-10份报关单证保管情况
  • 单证交接表/借阅登记表
  • 海关法律文书保管清单
  • 印章使用登记表、盘点记录
  • 机密文件销毁证明
🏢 现场：
  • 档案室门禁、标识
  • 电子单证归档系统截图
4. 禁限审查 (7) 赋分
(7) 禁止/限制性货物审查
有专门部门或岗位人员对进出境货物、物品进行禁止性或限制性管理审查，明确审查流程和处置要求。
📖 范围从"产品"扩到"货物、物品"，新增"审查流程和处置要求"。
📋 审核证据清单 📄 制度：
  • 《化学品进出口审批程序》
  • 出口管制合规审查流程
  • 审查流程和处置要求书面规定
📝 记录：
  • 化学品进口预审批记录
  • 出口管制筛查记录
  • 许可证件管理台账
👤 面谈：
  • 审查岗位人员能否说出主要法规依据
⚠️ 守法规范(20)将"违反出口管制"纳入处罚审查，与本条呼应。出口管制审查不力产生双重风险。
5. 信息系统 (8)-(9) 赋分
(8) 信息系统建设与流程管控
建立信息系统，对生产、采购、销售、物流、出入库、报关、税款缴纳、收付汇等环节实现流程检索跟踪，货物流、单证流、信息流互证。数据保存≥3年。ERP已与海关对接可免认证。
📋 审核证据清单 📄 制度：
  • 信息系统介绍文件（Oracle ERP、易网通等）
  • 数据备份政策
📝 记录：
  • 主要系统记录保存时间列表
  • 数据备份证明
🖥️ 系统演示：
  • 穿行测试：抽取一票报关单→追溯采购→入库→生产→出库→报关
  • 验证"三流互证"
  • 数据保存≥3年截图
✅ 免认证：ERP已与海关对接且满足管理要求。
(9) 数据安全与信息安全
建立数据安全、信息安全管理制度，使用防火墙、密码等措施，定期备份，防范内部恶意信息窃取、数据篡改。
📖 "数据安全"和"防范内部恶意窃取"是2026版新增。
📋 审核证据清单 📄 制度：
  • 《可接受使用政策》《企业密码政策》
  • 《网络安全政策》《数据备份政策》
  • IT机房管理程序
📝 记录：
  • 信息安全管控证明
  • 权限管理系统截图
  • 无重大信息安全事故申明
🏢 现场：
  • IT机房（温湿度、消防、UPS、门禁）
  • 防火墙配置截图
✅ 免认证（二选一）：① ISO 27001证书 ② 网络安全等级保护测评"符合"
6. 内部审计和改进 (10)-(11) 赋分
(10) 内部审计制度
每年对上一审计年度的进出口守法、规范情况以及持续符合高级认证标准开展内部审计，形成专门档案，按海关要求提交。
📋 审核证据清单 📄 制度：

  • AEO审计专员岗位职责

  • 审计工作底稿/抽样报关单明细
  • 完整审计报告（范围、发现、评价、结论、建议）
  • 审计调研表格、资料收集清单
  • AEO认证文件档案清单
  • 历年审计档案
(11) 问题处置和改进
未按规定报送年度报告、海关整改事项、违法情事、内审发现的问题，应由法定代表人或关务负责人组织处置和改进。

📝 记录：

  • 高管签署的整改方案
二、财务状况标准
7. 财务状况 (12)-(15) 赋分（三级判定）
(12) 审计报告
无保留意见=达标；保留意见=基本达标；否定/无法表示意见=不达标。每年6月30日前报送上年度审计报告。
📋 审核证据清单
  • 上年度审计报告（查看审计意见类型）
  • 审计报告报送海关记录（6月30日前）
  • 分支机构财务信息审计报告（如有备案）
⚠️ 否定/无法表示意见→直接不达标→认证不通过。
(13) 偿债能力
4项指标：①资产负债率≤85%(生产)/≤75%(非生产) ②现金比率≥0.2 ③经营现金流负债比≥0.1 ④流动比率≥1.0。2项及以上=达标；1项=基本达标；全不符=不达标。
📋 审核证据清单
  • 审计报告中的资产负债表、现金流量表
  • 4项偿债指标计算表（含公式和数据来源）
(14) 盈利能力
5项指标：①净利润＞0 ②营业利润率≥1% ③毛利率≥1% ④经营性现金流净额＞0 ⑤总资产报酬率≥1%。2项及以上=达标。
📖 2026版全新条款。
📋 审核证据清单
  • 审计报告中的财务报表数据
  • 5项盈利指标计算表
(15) 解散/破产
进入解散程序或被依法宣告破产的，视为不达标。
📋 审核证据清单
  • 营业执照（确认状态正常）
  • 国家企业信用信息公示系统查询截图
三、守法规范标准
8. 注册备案信息 (16) 仅达标/不达标
(16) 备案信息准确性
海关注册登记或备案信息与实际相符，2年内无未按规定办理变更手续的情形。
📖 2026版新增。常见风险：工商变更后忘记同步海关备案。
📋 审核证据清单
  • 营业执照
  • 海关注册登记证书
  • 近2年工商变更与海关变更对照表
  • 海关企业信用信息公示平台查询截图
9. 遵守法律法规 (17)-(20) 仅达标/不达标
(17) 刑事处罚
企业相关人员2年内未因故意犯罪被刑事处罚。
📖 范围从2022版4类人员扩大为"企业相关人员"，审查期从1年延至2年。
📋 审核证据清单
  • 相关人员无犯罪记录证明
  • 相关人员清单（姓名、职务、证明有效期）
(18) 单次处罚金额上限
1年内无被海关单次处罚超5万元的行为（未超上年度进出口总值万分之一的除外）。
📋 审核证据清单
  • 海关行政处罚信息查询截图
  • 1年内处罚明细
  • 上年度进出口总值数据
(19) 累计处罚次数和金额
1年内处罚次数≤3次或违法次数≤报关单票数千分之一，且累计金额≤10万元。
📋 审核证据清单
  • 同(18)处罚信息
  • 上年度报关单证总票数统计
(20) 禁限管理违法
2年内无因违反进口固体废物、出口管制相关禁限规定被处罚的情形。
📖 "出口管制"是2026版新增，零容忍标准。
📋 审核证据清单
  • 海关处罚查询（筛选固废和出口管制）
  • 出口管制合规管理记录
⚠️ 半导体/存储产业涉及敏感技术出口管制，一旦违规AEO资质可能丧失。
10. 业务开展 (21) 仅达标/不达标
(21) 进出口活动
1年内有进出口活动或开展的业务与海关直接相关。
📋 审核证据清单
  • 1年内进出口报关单统计（票数/金额）
11. 款项缴纳 (22) 仅达标/不达标
(22) 税款/罚款按时缴纳
2年内无超过法定期限未缴纳海关税款（含滞纳金）、罚款、没收违法所得的情形。
📋 审核证据清单
  • 海关滞纳税款情况查询
  • 罚款缴纳凭证
  • 税款缴纳记录
12. 申报规范 (23) 仅达标/不达标
(23) 报关单查获率
检查＜50票：查获≤3票且高质量查获率≤3%。检查≥50票：查获率≤10%且高质量查获率≤3%。
📖 2026版新增量化指标。
📋 审核证据清单
  • 海关查验情况统计（查验票数、查获票数、查获率）
  • 查验后整改记录
13. 税款要求 (24) 仅达标/不达标
(24) 追征税款限额
1年内累计追征税款≤10万元（超10万但≤上年度应纳税总额1%的除外）。
📖 2026版全新条款。
📋 审核证据清单
  • 海关追征税款记录
  • 估价补税情况
  • 上年度应纳税总额
14. 委托要求 (25)-(26) 仅达标/不达标
(25) 不得委托失信企业
2年内未委托失信/严重失信企业的报关企业办理报关手续。
📋 审核证据清单
  • 委托报关企业清单
  • 各报关企业信用等级查询截图
(26) 不得委托非备案企业
2年内未委托未在海关备案的报关企业或人员办理报关手续。
📋 审核证据清单
  • 报关委托书
  • 报关企业/人员备案查询截图
15. 管理要求 (27)-(36) 仅达标/不达标
(27) 行贿
1年内未向海关工作人员行贿。
📋 审核证据
  • 反腐败/反贿赂管理制度
  • 海关处罚/违法查询记录
(28) 提供虚假情况
1年内未向海关提供虚假情况或隐瞒重要事实。
📋 审核证据
  • 海关处罚记录查询
  • 申报差错跟踪记录
(29) 拒不配合执法
1年内未拒不配合海关执法活动。
📋 审核证据
  • 海关稽查/核查记录
(30) 抗拒阻碍执法
1年内未抗拒、阻碍海关工作人员依法执行职务。
📋 审核证据
  • 海关处罚记录查询
(31) 转移隐匿单证
1年内未转移、隐匿、篡改、毁弃报关单证。
📋 审核证据
  • 单证保管完整性检查
  • 海关处罚记录
(32) 拒绝提供材料
1年内未拒绝、拖延向海关提供相关材料。
📋 审核证据
  • 海关稽查/核查记录（配合情况）
(33) 逾期不改正
1年内未被海关责令限期改正但逾期不改正。
📋 审核证据
  • 海关责令改正文书及整改回复
  • 整改时限与完成时间对照
(34) 逾期不履行义务
1年内未由海关要求承担技术处理、退运、销毁义务但逾期不履行。
📖 2026版新增。
📋 审核证据
  • 海关退运/销毁通知及执行记录
(35) 信用信息异常名录
1年内未被列入海关信用信息异常企业名录。
📖 2026版新增。
📋 审核证据
  • 海关信用信息异常企业名录查询截图
(36) 虚增贸易统计处置
1年内未因涉嫌虚增贸易等原因被海关统计处置。
📖 2026版新增。
📋 审核证据
  • 海关统计处置记录查询
16. 海关信用 (37)-(39) 仅达标/不达标
